# cowork-productivity

25 AI-powered productivity skills for meetings, communication, projects, and content. Transform meeting chaos into actionable outcomes, generate project charters, track goals, and synthesize feedback.

## Installation

```bash
# Add the marketplace
/plugin marketplace add fabianaguilar/cowork-productivity

# Install the plugin
/plugin install cowork-productivity
```

Or test locally:

```bash
claude --plugin-dir /path/to/cowork-productivity
```

## Usage

Invoke any skill with:

```
/cowork-productivity:skill-name
```

For example:
```
/cowork-productivity:debrief-engine
```
